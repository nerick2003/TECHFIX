## üÜò Quick Troubleshooting

### Balance Sheet Doesn't Balance?
1. Check "Possible Issues" in Balance Sheet
2. Look for draft transactions in Journal
3. Post any draft entries
4. Regenerate balance sheet

### Transaction Not Showing?
- Check if it's saved as "draft"
- Check if it's in the correct period
- Check the transaction date

### Can't Find Account?
- Use the account code
- Check spelling
- Verify account is active

## üìä Understanding Your Balance Sheet

### What Should Balance:
```
Total Assets = Total Liabilities + Total Equity
```

### Example:
- Assets: ‚Ç±100,000
- Liabilities: ‚Ç±20,000
- Equity: ‚Ç±80,000
- Check: ‚Ç±100,000 = ‚Ç±20,000 + ‚Ç±80,000 ‚úÖ

### If It Doesn't Balance:
- Check "Possible Issues" section
- Most common: Draft transactions not posted
